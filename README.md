# pwa-template
This a base template for a Progressive Web Application.

This is just a skeleton structure for you to insert your code, css and js files and provide you with the prompt "add to homescreen" on mobile devices. 

******As long as your site is accessed over https******

#Directory Structure:

            |- index.html
            
            |- manifest.json
            
            |- service-worker.js
            
            |- styles/
            
            |- scripts/

              |- app.js

            |- img/    

                |- 1.jpg

                |- icons/
                  - * (all standard icons and favicons)       


index.html A starter HTML file with the supporting markup for your site
manifest.json A valid progressive web application web manifest file based on your choices in PWA Starter

favicon.ico A classic favicon.ico

readme.md A helpful reference document to help explain how to use the assets included in your PWA Starter zip file

sw.js A very basic service worker with install, active and fetch event handler placeholders

/a2hs Add to Homescreen library script and CSS

/meta Homescreen icons for different platforms and additional favicon images. Over 115 images in all
